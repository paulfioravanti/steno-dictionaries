# Punctuation Dictionary

This dictionary contains new and alternative outlines for entries that only
contain punctuation and/or non-word characters.

## New

```yaml
"*FPLT": "{^... }{-|}"
"H*F": "{^...?}{-|}"
"KR*T": "^{^^}"
"KW*GS": "{^\"}"
"KW*GSZ": "{^\"\"\"}"
"KW-GSZ": "{\"\"\"^}"
"PH-FP": "{^...?}{-|}"
"SKWO*ET": "{^'^}"
"TK*PL": "{^$^}"
"TK-RBS": "{^---^}"
"TK-RBZ": "{^---^}"
"TKHR*R": "{^$^}"
"TP-PL": "{^...^}"
```

## Alt

```yaml
"KW-GS": "{\"^}"
"KW-T": "{^\"^}"
"KWO*ET": "{^\"^}"
"OEUB": "{^\\^}"
"P-PS": "{^...}"
"PR-PB": "{^(^}"
"PWHRARB": "{^\\^}"
"PWR*BG": "{^]^}"
"SKP*D": "&&"
"SKWH-FBGS": "{^```^}"
"T*EULD": "{^~^}"
"TPR*BG": "{^\\}^}"  # [override]
"TPR*PB": "{^\\}^}"
"TPR-BG": "{^\\{^^}" # [override]
"TPR-PB": "{^\\{^^}"
```
